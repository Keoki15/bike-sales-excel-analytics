# Bike Sales Sorting and Filtering

## Project Overview

This workbook demonstrates how sorting and filtering can be used to organize a Bike Sales dataset and create focused views for analysis.

Sorting helps arrange records in a meaningful order, while filtering allows specific groups of records to be reviewed without changing or deleting the original data.

## Goal

The goal of this workbook was to organize the Bike Sales data so that important transactions, customer groups, products, and financial results could be located and compared more efficiently.

## Tasks Completed

- Sorted text values alphabetically
- Sorted numerical values from smallest to largest
- Sorted numerical values from largest to smallest
- Organized dates chronologically
- Applied multi-level sorting
- Filtered records by country and state
- Filtered records by customer characteristics
- Filtered products by category and subcategory
- Filtered revenue, cost, profit, and order quantity
- Created focused views based on selected conditions
