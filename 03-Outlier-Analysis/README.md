# Bike Sales Outlier Analysis

## Project Overview

This workbook demonstrates how Microsoft Excel can be used to identify unusually high or low values in a Bike Sales dataset.

Outliers may represent valid business events, exceptional sales performance, unusual customer behavior, or possible data-entry errors. Because outliers can affect calculations and business conclusions, they should be investigated before being removed.

## Goal

The goal of this workbook was to locate and review unusual observations that could affect the accuracy of the analysis.

[`Bike_Sales_Outlier_Analysis.xlsx`](./Bike_Sales_Outlier_Analysis.xlsx)

## Tasks Completed

- Reviewed numerical fields for unusually high or low values
- Compared individual records with the overall dataset
- Identified potential outliers in financial and sales fields
- Examined revenue, profit, cost, unit price, and order quantity
- Used sorting to locate extreme values
- Used visual and numerical comparisons to investigate unusual records
- Evaluated whether unusual values appeared valid or required further review

## Why Outlier Analysis Matters

Outliers can affect:

- Averages
- Totals
- Forecasts
- Charts and dashboards
- Statistical calculations
- Business recommendations
- Management decisions

An unusual value should not automatically be deleted. It should first be investigated to determine whether it is an error or a valid observation.
